pan-baidu-download
==================

## 使用方法：

    git clone git@github.com:banbanchs/pan-baidu-download.git
    cd pan-baidu-download
    ./panbaidu.py pan-baidu-url

------

## 实例

      ./panbaidu.py http://pan.baidu.com/share/link?uk=3943531277&shareid=4288212096
